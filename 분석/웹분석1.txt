1.ridibooks - 클릭시 홈으로 이동

2.ridiselect - 클릭시 ridiselect 페이지로 이동

3.검색창 - 엔터키로 검색가능, 클릭시 검색창 밑에 최근검색어 창이 뜸, 검색 시 도서 검색결과 페이지로 넘어감, 결과페이지는 분류가 기본 전체, 인기순으로 되어 있음

4.회원가입 - 회원가입 페이지로 이동 
	    헤더에 ridibooks글씨가 대문자로 클릭하면 홈으로 가는 링크 존재 
	    일반회원가입 클릭시 아이디, 비밀번호, 비밀번호확인, 이메일주소, 이름, 출생년도, 성별 약관 동의 선택창과 회원가입완료클릭버튼이 뜬다.
	    입력을 마치고 회원가입완료를 클릭시 인증메일발송을 했다는 페이지가 뜨고 메일에서 인증을 완료하면 회원가입을 축하한다고 창이 뜬다.

5.로그인 - 로그인 페이지로 이동
	 헤더에 ridibooks글씨가 대문자로 클릭하면 홈으로 가는 링크 존재 
	 아이디와 비밀번호를 입력할 수 있는 input태그 존재
	 그 밑에 로그인 상태유지란 체크박스 / 그 옆엔 아이디찾기와 비밀번호 재설정
	 그 밑엔 로그인과 회원가입 버튼이 존재 / 로그인 실패시 로그인버튼과 로그인상태체크박스 사이에 아이디 또는 비밀번호를 확인해주세요 빨간 글씨가 뜸
	 회원가입 버튼 클릭시 회원가입 페이지로 이동
	 아이디 찾기 클릭시 아이디찾기란 제목의 이메일주소입력 칸이 있고 찾기버튼이 있음
	 비밀번호 재설정 클릭시 비밀번호 재설정 제목의 아이디와 이메일주소 입력 칸이 있고 찾기 버튼이 있음

          - 일반 : 처음 들어간 홈 과 같은 페이지로 이동, 기본 선택되어져있다
          - 로맨스 : 클릭시 로맨스 장르의 웹소설/e북 선택가능한 페이지로 이동, hover시 글씨색이 연해짐, 글씨 검정색이다가 클릭하면 파란색으로 바뀜
          - 판타지 : 클릭시 판타지 장르의 웹소설/e북 선택가능한 페이지로 이동, hover시 글씨색이 연해짐, 글씨 검정색이다가 클릭하면 파란색으로 바뀜
          - 만화 : 클릭시 웹툰/e북 선택가능한 페이지로 이동, hover시 글씨색이 연해짐, 글씨 검정색이다가 클릭하면 파란색으로 바뀜
          - BL : 클릭시 BL 장르의 웹소설/소설e북/웹툰/만화e북 선택가능한 페이지로 이동, hover시 글씨색이 연해짐, 글씨 검정색이다가 클릭하면 파란색으로 바뀜
